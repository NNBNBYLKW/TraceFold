# Archive Guide

## Purpose

This directory keeps historical process evidence that is still worth retaining, but is no longer the default reading layer for current TraceFold work.

Enter archive when you need:

- step-by-step execution history
- historical acceptance evidence
- closeout-era checklists or recovery notes
- older task / round / chapter context behind a frozen boundary

Do not enter archive first when you only need the current baseline.

## About Execution-Era Task Materials

The current root-level docs surface has been trimmed so that execution-era task packs no longer stay in the main reading layer.

Those archived materials now include:

- older Step / Chapter / Round / Task materials under `docs/archive/phase1/**`
- the Post-Phase-1 Web PP1 execution pack and PP1 task docs under `docs/archive/post_phase1/web_pp1/`
